# AI-Powered Phishing Detection & Awareness System

A cybersecurity + machine learning project focused on detecting phishing emails and spreading awareness about social engineering attacks.

## Features

* Phishing email detection
* Suspicious link identification
* Risk classification
* Awareness guidelines
* Machine learning-based prediction

## Tech Stack

* Python
* Scikit-learn
* Streamlit
* Pandas
* Matplotlib
